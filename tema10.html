Sensor de Humedad y Temperatura

Objetivos

En este tema vamos a aprender cómo usar el sensor de humedad y temperatura DHT11. Este sensor es lo suficientemente exacto para la mayoría de los proyectos que necesitan hacer un seguimiento de las lecturas de humedad y temperatura.

Como en otros temas, vamos a usar una biblioteca diseñada específicamente para estos sensores que harán que nuestro código sea corto y fácil de escribir.

Introducción a los componentes

Sensor de humedad y temperatura

El sensor digital de temperatura y humedad DHT11 es un sensor compuesto que da una salida digital calibrada de la temperatura y la humedad. El sensor incluye un sensor resistente al agua y un dispositivo de medición de temperatura NTC, ambos conectados a un micro controlador de 8 bits de alto rendimiento.

Entre las aplicaciones mas comunes del sensor tenemos: HVAC, deshumidificador, bienes de consumo, controles automáticos, automóviles, registradores de datos, estaciones meteorológicas, electrodomésticos, regulador de humedad, etc.
