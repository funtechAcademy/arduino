Sensor de ultrasónico

Objetivos

El sensor ultrasónico es un componente ideal para todo tipo de proyectos que necesitan medidas de distancia, por ejemplo un robot que evita obstáculos.

En esta clase aprenderemos a usar el módulo ultrasónico HC-SR04 es barato y fácil de usar ya que vamos a usar una biblioteca diseñada específicamente para estos sensores.

Materiales

(1) x Elegoo Uno R3
(1) x Módulo de sensor ultrasónico HC-SR04
(4) x F M cables (cables de hembra a macho)

Introducción al componente

Sensor de ultrasónico

El módulo HC-SR04 de sensor ultrasónico proporciona una medición sin contacto de una distancia comprendida entre 2cm-400cm, la precisión varía entorno a los 3 mm. Los módulos incluyen transmisores ultrasónicos, receptor y circuito de control.
El principio básico del trabajo:

   1. Utilizando el disparador de ultrasonidos para lanzar una señal de nivel alto de al menos 10us.
   2. El Módulo una señal de 40 kHz y detecta si hay una señal que devuelve un pulso hacia atrás.
   3. Se calcula el tiempo que tarda en volver la señal para asi poder calcular la distancia.
